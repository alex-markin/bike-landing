## Installation

To get started with this project, follow these steps:

1. **Clone the repository:**

   git clone git@github.com:alex-markin/bike-landing.git 

   cd bike-landing

2. **Install dependencies:**

   npm install

## Usage

- **Start the dev server:**

  npm run start

- **Build the project:**

  npm run build
